# SmartCart Segmentation System

## Project Overview
SmartCart Segmentation System is an unsupervised machine learning project designed to analyze customer shopping behavior and group customers into different segments using clustering techniques.

## Features
- Customer behavior analysis
- Customer segmentation
- Data visualization
- PCA-based dimensionality reduction
- Cluster visualization

## Machine Learning Concepts Used
- Unsupervised Learning
- K-Means Clustering
- PCA (Principal Component Analysis)
- Data Standardization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Dataset
Customer shopping and purchase behavior dataset.
