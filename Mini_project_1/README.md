# CreditWise Loan System

## Project Overview
CreditWise Loan System is a supervised machine learning project developed to predict whether a loan application should be approved or rejected based on applicant information and financial details.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Loan approval prediction
- Model training and evaluation
- Accuracy analysis

## Machine Learning Concepts Used
- Supervised Learning
- Classification Algorithms
- Train-Test Split
- Feature Selection

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Dataset
Loan approval dataset containing applicant and financial information.
